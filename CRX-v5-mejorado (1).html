Paso 1 — Crear la hoja de cálculo en Google Drive

  1. Abre drive.google.com con tu cuenta de Google.
  2. Haz clic en + Nuevo → Hojas de cálculo de Google.
  3. Nómbrala: CRX_Radiologia_DB
  4. En la hoja verás la pestaña "Hoja 1" al fondo. Renómbrala a Ventas (doble clic sobre el nombre).
  5. Crea 3 hojas adicionales haciendo clic en el + y nombra cada una: Inventario, Recetas, Gastos.
  6. En la hoja Ventas, escribe en la fila 1 estos encabezados (cada uno en su propia columna A→P):
    ID_Venta | Fecha | Nombre | ID_Paciente | Genero | Telefono | Email | Direccion | Nacimiento | Edad | Clinica | Estudio | Formato | Metodo_Pago | Valor | Notas
  7. En la hoja Inventario: ID | Nombre | Stock | PuntoReorden
  8. En la hoja Recetas: ID | Estudio | ArticuloID | ArticuloNombre | Cantidad
  9. En la hoja Gastos: ID | Fecha | Concepto | Monto | Comprobante
💡 Guarda la URL de la hoja. La necesitarás en el Paso 4. Tiene esta forma: https://docs.google.com/spreadsheets/d/TU_ID_AQUI/edit
⚙️

Paso 2 — Crear el Apps Script (backend gratuito)

  1. Con la hoja abierta, ve al menú Extensiones → Apps Script.
  2. Se abrirá el editor. Borra todo el código que hay y pega este:
const SS = SpreadsheetApp.getActiveSpreadsheet(); // ── Cabeceras CORS — necesarias para fetch desde GitHub Pages ── function setCORS(output) { return output .setMimeType(ContentService.MimeType.JSON); // Nota: Apps Script maneja CORS automáticamente cuando el script // está publicado como "Cualquier usuario (incluso anónimos)" } function doPost(e) { try { const raw = e.postData ? e.postData.contents : '{}'; const data = JSON.parse(raw); const sheet = SS.getSheetByName(data.sheet); if (!sheet) return res({ok:false, error:'Hoja no encontrada: '+data.sheet}); if (data.action === 'append') { sheet.appendRow(data.row); return res({ok:true}); } if (data.action === 'overwrite') { const lastRow = sheet.getLastRow(); if (lastRow > 1) sheet.deleteRows(2, lastRow - 1); if (data.rows && data.rows.length > 0) { data.rows.forEach(r => sheet.appendRow(r)); } return res({ok:true, written: data.rows ? data.rows.length : 0}); } if (data.action === 'read') { return res({ok:true, data: sheet.getDataRange().getValues()}); } return res({ok:false, error:'Acción desconocida: '+data.action}); } catch(err) { return res({ok:false, error:err.toString()}); } } function doGet(e) { try { const sheetName = e.parameter && e.parameter.sheet; if (sheetName) { const sheet = SS.getSheetByName(sheetName); if (!sheet) return res({ok:false, error:'Hoja no encontrada: '+sheetName}); const values = sheet.getDataRange().getValues(); return res({ok:true, data: values, rows: values.length}); } // Ping de prueba de conexión return res({ok:true, msg:'CRX API v4 activa ✓', sheets: SS.getSheets().map(s=>s.getName())}); } catch(err) { return res({ok:false, error:err.toString()}); } } function res(obj) { return ContentService .createTextOutput(JSON.stringify(obj)) .setMimeType(ContentService.MimeType.JSON); }
💡 Este script convierte tu hoja en una API gratuita. La clave para que funcione desde GitHub Pages es publicarlo como "Cualquier usuario" en el Paso 3.
🚀

Paso 3 — Publicar el script como aplicación web

  1. En el editor de Apps Script, haz clic en Guardar (Ctrl+S o ícono de disquete).
  2. Haz clic en el botón azul "Implementar" → "Nueva implementación".
  3. En "Tipo" selecciona Aplicación web.
  4. Configura exactamente así (cualquier otra opción causará error CORS):
    ✓Descripción: CRX API v4
    ✓Ejecutar como: Yo (tu correo de Google)
    ⚠Quién tiene acceso: Cualquier usuario (incluso anónimos) ← CRÍTICO
  5. Clic en Implementar. Google pedirá autorización — acepta todos los permisos.
  6. Copia la URL de la aplicación web:
    https://script.google.com/macros/s/XXXXXXXXXXXXXXXXXXXXXXXX/exec
⚠️ Si cambiaste el código después: debes crear una nueva implementación (no editar la existente). Clic en Implementar → Administrar implementaciones → Nueva → copiar la nueva URL.

🔍 Si "Probar conexión" da error CORS: el acceso NO está en "Cualquier usuario". Crea una nueva implementación con el acceso correcto y actualiza la URL en la app.
